# JavaScript Quiz App

A simple and interactive quiz application built with HTML, CSS, and JavaScript. The app lets users take a timed quiz, answer questions, track their score, and resume progress later.

## Features

- Interactive quiz experience with multiple-choice questions
- Randomized question order for variety
- Randomized answer options for each question
- Countdown timer for each question
- Score tracking and final result screen
- High score saved in browser storage
- Pause/resume functionality for unfinished quizzes
- Progress saved locally so users can return later
- Responsive layout for different screen sizes
- Keyboard-friendly navigation and focus states
- Accessible button labels and screen-reader-friendly structure

## What Makes This Project Unique

This quiz app goes beyond a basic quiz by including:

- Resume support after pausing or refreshing the page
- Persistent quiz state using browser storage
- A cleaner, more accessible user experience
- Better keyboard navigation for users who do not use a mouse
- A more polished UI with improved button and focus styling

## Project Structure

- index.html — Main structure of the quiz app
- style.css — Styling for layout, buttons, quiz cards, and responsive design
- javascript.js — Quiz logic, timers, scoring, pause/resume, and local storage

## How It Works

1. The user opens the app and clicks Start Quiz.
2. Questions appear one at a time with randomized answer choices.
3. The timer counts down for each question.
4. The user selects an answer and moves to the next question.
5. At the end, the app shows the final score and high score.
6. If the user pauses or refreshes, the app can resume from the saved state.
